# interview-docs

嵌入式面试准备文档仓库，用于在不同设备之间同步规划、复盘、简历和项目材料。

当前定位：

- 记录长期面试准备规划。
- 汇总阶段性学习复盘。
- 管理简历和项目表达材料。
- 沉淀投递与面试反馈。

具体知识点训练建议放在独立对话窗口或独立文档中，再将结论同步回本仓库。

## 长期项目结构

```text
00-control/
  strategy.md              # 总体策略、阶段目标、投递方向

01-profile/
  career-profile.md        # 个人基础档案、项目概况、约束条件

90-assistant-notes/
  evidence-map.md          # 助手工作笔记：证据链、风险、待确认问题
```

后续根据需要再逐步增加：

```text
02-projects/               # 项目总览与面试叙事，不放大段代码
03-resume/                 # 主简历和岗位定制版本
04-reviews/                # 周复盘、模拟面试反馈、投递反馈
05-jd/                     # JD 检索、岗位匹配和投递记录
```
